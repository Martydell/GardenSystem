# 🌿 Garden Plant Catalogue

A self-contained, browser-viewable catalogue of my plant collection —
photos, origin, care instructions, propagation methods, hardiness,
classification, sowing/flowering times, and time to maturity for each plant.

## Usage
Open `index.html` in any browser. No install, no server, no dependencies —
photos are embedded directly in the file.

Features:
- Live search across all plant data
- Filter by category (Edible / Medicinal / Ornamental / Toxic / etc.)
- Filter by upload batch

## Status
See [`CONTEXT.md`](./CONTEXT.md) for full project history, identification
notes, and open items — especially which plants still need ID confirmation
(marked ⚠ in the catalogue itself).

## Updating
This catalogue was built iteratively with Claude, batch by batch, from photos
of the plants. To continue: open a new chat with Claude, attach `CONTEXT.md`,
and upload the next batch of photos.
